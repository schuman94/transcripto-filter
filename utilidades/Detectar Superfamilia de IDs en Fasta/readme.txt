Este script crea un excel en donde tenemos una columna con identificadores de secuencias y otra columna con la superfamilia a la que pertenece.

Como entrada se utiliza un fichero fasta con todas las secuencias de las que se quiere obtener el ID.
Tambien se utiliza el fichero txt con la base de datos de secuencias de conotoxinas anotadas (virro).

El script extrae el ID del fichero fasta y busca en el fichero txt dicho ID, seleccionando la superfamilia a la que pertenece.
